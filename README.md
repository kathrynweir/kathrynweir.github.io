# Kathryn Weir website

This is the Jekyll version of the site.

## Editing the site

- Shared page shell: `_layouts/default.html`
- Header navigation: `_includes/nav.html`
- Footer: `_includes/footer.html`
- Image include: `_includes/image.html`
- Global image alt text: `_data/image_alts.yml`
- Main styling: `assets/css/style.css`

Every image included through `_includes/image.html` gets its alt text from `_data/image_alts.yml`.
